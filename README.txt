Paul Gasbarra — Personal Portfolio
====================================
https://paulgasbarra.github.io

A personal portfolio and resume site for Paul Gasbarra, senior software engineer
based in New York. Built to be fast, clean, and easy to maintain.


Tech
----
- Plain HTML and CSS — no build step, no framework
- Deployed via GitHub Pages
- Fonts: Playfair Display + Jost (Google Fonts)

---------
index.html          Main page
assets/css/site.css All styles
images/             Avatar, favicon, and company logos
*.pdf               Resume
